MoviesOnStreamingPlatforms: Description back to ToC

This dataset provides us with extra information on both Netflix and Disney+ movies.

#### R2RML Mapping for Walt Disney dataset ```xml @prefix rr: . @prefix ex: . @prefix xsd: . @prefix rdf: . @prefix rmlc: . <#Dataset4TripleStore> a rr:TriplesMap; rr:logicalTable [ rr:sqlQuery """ SELECT dataset_4.*, CASE WHEN (NETFLIX = 1) THEN 'Netflix' WHEN (DISNEYPLUS = 1) THEN 'DisneyPlus' END AS PROVIDER FROM dataset_4 """ ]; rr:subjectMap [ rr:template "http://www.example.org/entertainment/Movie/{TITLE}"; ]; rr:predicateObjectMap [ rr:predicate rdf:type; rr:objectMap [rr:template "http://www.example.org/entertainment/Movie"]; ]; rr:predicateObjectMap [ rr:predicate ex:title; rr:objectMap [ rr:column "TITLE" ]; ]; rr:predicateObjectMap [ rr:predicate ex:provider; rr:objectMap [rr:template "http://dbpedia.org/resource/{PROVIDER}"]; ]; rr:predicateObjectMap [ rr:predicate ex:director; rr:objectMap [ rr:column "DIRECTORS" ]; ]; rr:predicateObjectMap [ rr:predicate ex:country; rr:objectMap [ rr:column "COUNTRY" ]; ]; rr:predicateObjectMap [ rr:predicate ex:imdbRating; rr:objectMap [ rr:column "IMDB"; rr:datatype xsd:float ]; ]; rr:predicateObjectMap [ rr:predicate ex:rottenTomatoesRating; rr:objectMap [ rr:column "Rotten Tomatoes"; rr:datatype xsd:float ]; ]; rr:predicateObjectMap [ rr:predicate ex:duration; rr:objectMap [ rr:column "RUNTIME"; rr:datatype xsd:integer ]; ]; rr:predicateObjectMap [ rr:predicate ex:genres; rr:objectMap [ rr:column "GENRES" ]; ]. ```
